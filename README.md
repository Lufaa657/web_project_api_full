# Tripleten web_project_api_full

# Descripción

web_project_around_react es una aplicación web interactiva desarrollada con React. Permite a los usuarios visualizar una galería de tarjetas con diferentes lugares, agregar nuevas tarjetas, eliminar las existentes y dar "like" a sus favoritas.

Además, los usuarios pueden editar su perfil y actualizar su avatar. La aplicación se conecta a una API para gestionar y persistir los datos en el servidor.

# Características principales

- Visualización de tarjetas  
- Agregar nuevas tarjetas  
- Eliminar tarjetas  
- Dar y quitar "like"  
- Editar perfil  
- Actualizar avatar  
- Validación de formularios  
- Conexión con API REST
